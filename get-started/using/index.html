USING BITCOIN NOVA

Transacting with Bitcoin Nova can be made easy. This page is designed to guide users in that process.

1. Learn


Bitcoin Nova is a secure, private, and untraceable cryptocurrency. The developers and community are committed to protecting these values. Learn more by reading the What is Bitcoin Nova page. The source code is also available for review and discussion.

2. Request Support


There is a large and supportive community that will assist if you experience any difficulty. See the Hangouts page for more information.

3. Generate a Wallet


A Bitcoin Nova wallet is required to secure your own funds. See the Downloads page for a listing of available wallets.
The easiest way to run a Bitcoin Nova node, without affecting your home bandwidth, is to purchase a VPS (Virtual Private Server).

4. Get Bitcoin Nova


Bitcoin Nova may be purchased on an exchange with fiat or other cryptocurrencies. An alternate way of acquiring Bitcoin Nova is via mining, the computationally-complex process whereby transactions are immutably recorded on the blockchain.

5. Send and Receive Bitcoin Nova


Learn how to send and receive Bitcoin Nova by viewing the guide.

6. Transact with Bitcoin Nova


Bitcoin Nova may be used to purchase many goods and services. For a listing, see the Merchants page.
